Code examples

Calling from an event

To trigger an action use a change event.

{% highlight js %}$(':checkbox').checkboxpicker().on('change', function() { ... });{% endhighlight %}

Or call it, by an specific id member of the DOM.

{% highlight js %}$('#input-1').checkboxpicker(); $('#input-1').on('change', function() { ... });{% endhighlight %}

Changing checkbox value

Change checkbox checked value.

{% highlight js %}$('#input-1').prop('checked', false);{% endhighlight %}

Disabling checkbox value

Disable checkbox.

{% highlight js %}$('#input-1').prop('disabled', true);{% endhighlight %}
